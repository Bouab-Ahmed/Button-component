

# Button component

A reuseable button component implemented in react



## Demo

https://bouab-ahmed.github.io/Button-component/

## Installation

clone the repository

```bash
  git clone https://github.com/Bouab-Ahmed/Button-component.git
```
cd the project 

```bash
  cd Button-component
```
install Dependencies nad run the project

```bash
  npm install
  npm start
```
